一、功能概述

1.1 定义

液体神经网络 (Liquid Neural Network, LNN) 是每只蛊虫的初始智能核心，具有动态拓扑结构、连续时间处理和自适应学习能力。它是蛊虫"思考"的基础设施。

1.2 核心特性

特性 说明 与传统神经网络对比
动态拓扑 网络结构可随任务需求动态变化 传统网络结构固定
连续时间 基于微分方程，时间连续流动 传统网络离散时间步
自适应学习 根据反馈自主调整学习策略 传统网络固定学习率
小样本学习 少量样本即可适应新任务 传统网络需要大量数据
可解释性 每个神经元状态可追踪解释 传统网络是黑盒

二、设计原则

2.1 核心原则

🧠 智能涌现

智能不是被编程的，而是从简单规则中涌现。LNN提供基础机制，能力从学习中生长。

🔄 动态适应

网络结构不是静态的，而是根据任务需求动态调整。复杂任务自动扩展，简单任务自动简化。

⚡ 实时响应

连续时间处理使蛊虫能够实时响应环境变化，而不是等待离散的时间步。

🔐 安全可控

动态变化必须有边界约束，确保网络演化不会失控。

2.2 设计约束

⚠️ 安全约束（螺丝咕姆要求）：
  • 网络规模有上下限（防止资源耗尽或能力不足）
  • 拓扑变化需要审计记录
  • 异常行为触发熔断机制
⚡ 优雅约束（拉蒂奥要求）：
  • 避免过度复杂化，保持最简有效结构
  • 拓扑变化应该对称、平衡

四、液体神经元设计

4.1 神经元模型

核心思想：液体神经元的状态随时间连续变化，通过微分方程描述其动态行为。

4.1.1 状态方程

τ · (dx/dt) = -x + Σ(wᵢⱼ · σ(xⱼ)) + I(t)
符号 含义 说明
x 神经元状态 随时间连续变化
τ 时间常数 控制响应速度，可学习
wᵢⱼ 突触权重 连接强度，动态可变
σ 激活函数 非线性变换
I(t) 外部输入 时间相关的输入信号

4.1.2 神经元属性

属性 类型 范围 说明
状态值 (state) float [-1, 1] 当前激活状态
时间常数 (tau) float [0.1, 10.0] 响应速度控制
偏置 (bias) float [-1, 1] 激活阈值调整
活跃度 (activity) float [0, 1] 近期使用频率
重要度 (importance) float [0, 1] 对输出的贡献度

4.2 神经元类型

🔹 感知神经元

接收外部输入，负责感知环境信息。

  • 高时间常数（慢响应）
  • 对输入信号敏感
  • 固定数量，不可删除

🔸 认知神经元

处理信息，负责推理和记忆。

  • 可变时间常数
  • 动态增删
  • 核心计算单元

🔹 行为神经元

产生输出，负责行为决策。

  • 低时间常数（快响应）
  • 对输出直接负责
  • 数量根据任务动态调整

六、自适应学习机制

6.0 LNN vs Transformer学习方式的根本区别

⚠️ 重要区分：

LNN和Transformer的学习方式完全不同：

特性 Transformer (反向传播) LNN (局部学习)
学习方式 全局梯度下降，需要完整计算图 局部规则，每个神经元独立调整
更新时机 批量训练后一次性更新 实时、连续、在线更新
信息流向 前向传播 → 反向传播（反向） 只有前向流动，无反向传播
计算需求 需要存储中间状态用于反向传播 只需当前状态，内存友好
生物学合理性 低（大脑没有反向传播） 高（接近生物神经元）

6.1 LNN的核心学习规则

LNN使用局部学习规则，而非梯度下降：

6.1.1 赫布学习 (Hebbian Learning)

Δw = η · xᵢ · xⱼ "一起激活的神经元，连接在一起"

解释：如果神经元i和神经元j同时活跃，它们之间的连接权重增加。这是最基础的局部规则。

6.1.2 STDP (脉冲时序依赖可塑性)

如果 i 先于 j 激活: Δw = +A · exp(-Δt/τ₊) 如果 i 后于 j 激活: Δw = -B · exp(-Δt/τ₋) Δt = tⱼ - tᵢ (时间差)

解释：时序关系决定权重变化方向。如果神经元i先于j激活，说明i可能导致j激活，连接增强；反之减弱。

6.1.3 Oja规则 (防止权重爆炸)

Δw = η · y · (x - y · w) 其中 y = w · x (输出)

解释：赫布学习的改进版，加入了权重衰减项，防止权重无限增长。

6.3 情绪调节学习

蛊虫特色：LNN的学习率可以由情绪系统调节
η' = η · g(emotion) 其中: - 高唤醒情绪（兴奋、恐惧）→ 学习加速 - 低唤醒情绪（平静、放松）→ 学习放缓 - 负价情绪（愤怒、悲伤）→ 可能触发快速适应

6.4 学习策略（非梯度下降）

策略 触发条件 学习规则调整
快速适应 紧急情况、新任务 增大η，加速权重变化
稳定学习 已掌握任务 减小η，保持稳定
探索学习 未知领域 加入随机扰动到权重
知识迁移 相似任务 复用已有拓扑结构

6.5 LNN不需要梯度下降的原因

✅ 局部信息足够

每个神经元只需要知道自己的输入和输出，就可以调整权重。不需要知道整个网络的误差。

✅ 连续时间处理

LNN在连续时间中运行，每个时刻都在学习和调整，不存在"训练阶段"和"推理阶段"的分离。

✅ 任务驱动学习

蛊虫通过完成任务获得反馈（金币奖励等），这个反馈信号可以直接调节学习率，不需要损失函数。

✅ 自适应拓扑

LNN可以动态增删神经元和连接，这是反向传播难以处理的。
