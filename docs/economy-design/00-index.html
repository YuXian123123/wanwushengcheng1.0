防刷金币机制

五层防护： 内容质量 → 相似度检测 → 行为分析 → 信任评分 → 审计追踪

系统通过五层防护机制防止刷金币行为，每层都有动态阈值和验证公式。 相似度检测要求内容相似度不超过80%，阈值基于发送者历史动态调整。

物竞天择原则

  • 无寿命延长限制：蛊虫有自然寿命，无法无限延长
  • 生存压力传导：压力随时间指数增长
  • 自然选择：优胜劣汰，适者生存
  • 欲望驱动：金币欲望推动能力进化
