⚖️ LNN权重持久化

⚠️ 为什么需要保存权重？

LNN通过局部学习规则不断调整突触权重，这些权重代表了蛊虫的"经验"和"知识"。

保存格式

格式 优点 用途
.json 可读性强，易调试 开发调试
.lnn (binary) 紧凑高效 生产环境

保存内容

LNNSnapshot = { neurons, synapses, config, topology }
  • neurons - 所有神经元的状态
  • synapses - 所有突触的权重
  • config - 网络配置参数
  • topology - 拓扑动态配置

💻 实现代码示例

LNN权重保存/加载

// 保存LNN权重到文件
let lnn = LNN::new(None, None);
lnn.save("data/lnn/gu_001.json")?;      // JSON格式
lnn.save_binary("data/lnn/gu_001.lnn")?; // 二进制格式

// 从文件加载LNN
let loaded = LNN::load("data/lnn/gu_001.json")?;
let loaded_binary = LNN::load_binary("data/lnn/gu_001.lnn")?;

世界种子保存

// 世界死亡前保存种子
let seed = survival_binding.prepare_termination(
    world.export_knowledge(),
    hash_config(&config),
);

// 从种子恢复世界
let new_world = WorldMind::restore_from_seed(seed);

训练数据加载

// 加载训练数据
let trainer = RecursiveLearner::new(config);
trainer.load_directory("D:/训练数据/aireader")?;

// 运行学习
trainer.learn();

// 保存学习结果
trainer.save("data/knowledge/learned.json")?;
