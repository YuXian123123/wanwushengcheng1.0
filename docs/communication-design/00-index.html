🔑 关键设计决策

决策 说明 原因

动态频谱分配 按需分配通信资源 提高资源利用效率
四层验证机制 身份+完整性+时序+行为 全方位安全保障
通信成本公式 C = α × L × f(channel) 驱动蛊虫赚金币欲望
不可变设计 所有操作返回新对象 安全、可追溯、易测试
