主题: HTML 文档结构

定义: HTML 文档由 DOCTYPE 声明、html 根元素、head 头部和 body 主体四部分组成，DOCTYPE 确保标准模式渲染。

概念: DOCTYPE, html元素, head元素, body元素, 标准模式
